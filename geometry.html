Points vs Vectors

In 2-dimensional space (the xy-plane) and 3-dimensional space (xyz-space), the terms "point" and "vector" refer to the same thing. The reason different terminology is used is that when someone refers to a "point" as a vector, they have in mind an arrow that points from the origin (the point whose coordinates are all zero) to that point.

Distance between two points

We will write points as arrays. So for instance a point \((2, 3, 4)\) will be written as an array [2, 3, 4]. To calculate the distance between two points, use the following function.

To compute the distance between two points, use the following.

distance function

var distance = function(u, v) {
  const dim = u.length;          
  let sumSquares = 0;
  for (let i = 0; i < dim; i++)
    sumSquares += (u[i] - v[i]) ** 2;
  return Math.sqrt(sumSquares);
}

The inputs u and v should be arrays of numbers and they should have the same number of entries. See below for how to use it.

Example:

Let c and d be three-dimensional points (points in xyz-space) where the x-coordinate of c is 1.2, the y-coordinate of c is 4.4, the z-coordinate of c is 0.3, the x-coordinate of d is 3.1, the y-coordinate of d is 3.6, and the z-coordinate of d is -3.2. Calculate the distance between c and d.

Answer

To calculate the distince between c and d, do the following.
