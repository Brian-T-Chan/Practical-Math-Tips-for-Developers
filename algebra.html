Writing formulas in code (Order of Operations).

Make sure that the brackets are in the right positions. When there are no brackets, perform multiplication and division first, then addition and subtraction.

Example 1

Set

var funct1 =

(x, y, z) => x + y * z;

var funct2 =

(x, y, z) => x + (y * z);

var funct3 =

(x, y, z) => (x + y) * z;

Then funct1 outputs the same answers as funct2, but funct3 outputs different answers than funct1 and funct2. For instance, both of the following

funct1(4, 1, 5) ===

funct2(4, 1, 5);

funct1(4, 1, 5) !==

funct3(4, 1, 5);

evaluate to true.

Example 2

Set

var funct4 =

(x, y, z) => x/y + z;

var funct5 =

(x, y, z) => x/(y + z);

Then funct4 outputs different answers than funct5.

Example 3

Set

var funct6 = (w, x, y, z)

=> v - w * x / y + z;

var funct7 = (w, x, y, z)

=> v - (w * x / y) + z;

Then funct6 outputs the same answers as funct7.

Example 4

Set

var funct8 = (v, w, x, y, z)

=> v / z - w * z

+ x / z - y / z;

var funct9 = (w, x, y, z)

=> (v / z) - (w * z)

+ (x / z) - (y / z);

Then funct8 outputs the same answers as funct9.

Example 5

Set

var funct4 = (w, x, y, z)

=> w + x/(y + z);

var funct5 = (w, x, y, z)

=> (w + x)/(y + z);

Then funct4 and funct5 will output different answers.


Sometimes, brackets are not needed.

Example 6

Set

var funct6 = (x, y, z)

=> x * (y / z);

var funct7 = (x, y, z)

=> (x * y)/z;

Then funct6 and funct7 will output the same answers.

Example 7

Set

var funct8 = (x, y, z)

=> x + (y - z);

var funct9 = (x, y, z)

=> (x + y) - z;

Then funct8 and funct9 will output the same answers.


Sometimes, we have to be careful when using brackets with subtraction and division.

Example 8

Set

var funct10 = (x, y, z)

=> x - (y - z);

var funct11 = (x, y, z)

=> x - y - z;

var funct12 = (x, y, z)

=> x - y + z;

Then funct10 and funct11 will output different answers, but funct10 and funct12 will output the same answers.

Example 9

Set

var funct13 = (x, y)

=> 1 / (x / y);

var funct14 = (x, y)

=> (1 / x) / y;

var funct15 = (x, y)

=> y / x;

var funct16 = (x, y)

=> 1 / (x * y);

Then funct13 and funct15 output the same answers, funct14 and funct16 output the same answers, and funct13 and funct14 output different answers.



The distributive law:

Multiplication and division spread into addition and subtraction.

Set

var functa = (x, y, z)

=> x * (y + z);

var functb = (x, y, z)

=> (x * y) + (x * z);

Then functa outputs the same answers as functb. This fact, that functa and functb give the same answers, is the distributive law. Reformulations of this law are also true.

Example 10

Set

var functc = (x, y, z)

=> (y - z) * x;

var functd = (x, y, z)

=> (y * z) - (z * x);

Then functc outputs the same answers as functd.

Example 11

Set

var functe = (x, y, z)

=> (y - z + w) / x;

var functf = (x, y, z)

=> (y / x) - (z / x) + (w / x);

Then functe outputs the same answers as functf.



Exponents

Exponents spread into multiplication and division.

Example 12

Set

var functg = (x, y, z)

=> (x / y) ** z;

var functh = (x, y, z)

=> (x ** z) / (y ** z);

var functi = (x, y, z)

=> x ** z / y ** z;

Then functg, functh, and functi output the same answers. functh outputs the same answers as functi because exponents come before multiplication and division. functg outputs the same answers as functh because exponents spread into multiplication and division.

Example 13

Set

var functj = (x, y, z)

=> (x * y) ** z;

var functk = (x, y, z)

=> (x ** z) * (y ** z);

var functl = (x, y, z)

=> x ** z * y ** z;

Then functj, functk, and functl output the same answers.


Negative Exponents

Negative exponents are a combination of exponents and division.

Example 14

Set

var functm = (x, y)

=> x ** -y;

var functn = (x, y)

=> 1 / (x ** y);

Then functm and functn output the same answers.


Square roots

Square roots are useful for tasks like computing distances. The square root also spreads into multiplication and division.

Example 15

Set

var functo = (x, y)

=> Math.sqrt(x * y / z);

var functp = (x, y)

=> Math.sqrt(x) * Math.sqrt(y) / Math.sqrt(z);

Then functo and functp output the same answers.



Sample Problem

Sally buys a car and a truck. The truck is 1000 USD and the car, normally priced at 500 USD, is at a 30% discount. How much did Sally pay?


Answer

We know Sally has to pay 1000 USD for the car, so we can set

var truckPrice = 1000;

We need to know how much Sally needs to pay for the car. Since the car is normally priced at 500 USD. So we can set

var carPrice = 500;

How much does Sally save due to the above discount? The discount is 30%, so we can set

var discount = 30;

So Sally saves 150 USD.

carPrice * discount / 100 === 150

Because of that, Sally pays 350 USD

carPrice - (carPrice * discount / 100) === 350

By the distributive law

carPrice - (carPrice * discount / 100) ===

carPrice * (1 - discount / 100)

is true. Note that

1 - discount / 100 ===

1 - (discount / 100)

is true. So Sally pays 1350 USD as

truckPrice + carPrice * (1 - discount / 100) === 1350

is true.
